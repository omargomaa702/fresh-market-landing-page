
# 🛒 Fresh Market Landing Page

A clean and responsive landing page for a fresh food store.
It displays products like vegetables, fruits, and meat with a simple add-to-cart interface.

---

## 🚀 Features

* Responsive design
* Product cards layout
* Add to cart buttons
* Clean and modern UI

---

## 🛠️ Technologies Used

* HTML
* CSS
